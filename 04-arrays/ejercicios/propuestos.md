# Ejercicios Propuestos – Arrays en Java

## Ejercicio propuesto 1 – Ingreso de cadenas por teclado

**Enunciado**

Crear un programa en Java que:

- Solicite al usuario la cantidad de nombres a ingresar
- Cree un arreglo de tipo String con ese tamaño
- Ingrese cada nombre por teclado
- Solicite un nombre a buscar
- Muestre cuántas veces aparece ese nombre en el arreglo

⚠️ La comparación debe realizarse correctamente para cadenas.

---

## Ejercicio propuesto 2 - Contar números primos en un arreglo

Escriba un programa en Java que:

- Solicite al usuario el tamaño de un arreglo de números enteros.
- Permita ingresar los valores del arreglo por teclado.
- Determine cuántos números primos hay dentro del arreglo.
- Muestre el arreglo y la cantidad total de números primos encontrados.

Recuerde que un número primo es aquel que solo es divisible por 1 y por sí mismo.

---

## Ejercicio propuesto 3 – Nombre más largo

Enunciado

Dado un arreglo de cadenas que contiene nombres de personas:

```java
String[] nombres = {"Carlos", "Ana", "Sebastian", "Luis", "Maria"};
```

Crear un programa que:

- Recorra el arreglo
- Determine cuál es el nombre con mayor cantidad de caracteres
- Muestre el nombre y su longitud

Si hay más de uno con la misma longitud, mostrar el primero que aparezca.

---

## Ejercicio propuesto 4 – Conteo por inicial

**Enunciado**

Crear un programa que:

- Solicite al usuario ingresar 5 palabras
- Las almacene en un arreglo de tipo String
- Solicite una letra por teclado
- Cuente cuántas palabras comienzan con esa letra 
- Muestre el resultado

⚠️ El programa debe funcionar correctamente sin distinguir entre mayúsculas y minúsculas.

---

## Ejercicio propuesto 5 – Ordenamiento por método burbuja

**Enunciado**

Crear un programa en Java que permita ordenar un arreglo de números enteros utilizando el **método de ordenamiento burbuja**.

El método burbuja consiste en recorrer el arreglo varias veces, comparando elementos consecutivos e intercambiándolos cuando están en el orden incorrecto, hasta que el arreglo quede completamente ordenado.

**Requisitos**

- Declarar un arreglo de números enteros con al menos 6 valores desordenados
- Implementar el algoritmo burbuja usando ciclos anidados
- Ordenar el arreglo en orden ascendente
- Mostrar el arreglo antes y después del ordenamiento
- No utilizar métodos de ordenamiento predefinidos
