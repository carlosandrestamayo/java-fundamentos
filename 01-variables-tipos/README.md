# Unidad 1 – Variables y Tipos de Datos en Java
## Descripción

Esta unidad introduce los conceptos fundamentales de Java relacionados con:

- Variables y su declaración
- Tipos de datos primitivos y no primitivos
- Diferencia entre valor y referencia
- Alcance (scope) y tiempo de vida de variables
- Operaciones básicas con variables

El objetivo es que el estudiante pueda crear, asignar y manipular variables, comprendiendo cómo funcionan en memoria y cómo se comportan los tipos de datos.

---

### Contenido de la Unidad

1. **Teoría** – Conceptos y explicaciones:

    - Variables y tipos de datos
    - Valores por defecto
    - Diferencia entre tipos primitivos y referencias
    - Copia por valor vs copia por referencia

2. **Ejercicios prácticos** – Para reforzar los conceptos:

    - Declaración y asignación de variables
    - Copia de valores y referencias
    - Operaciones aritméticas básicas
    - Tipos de datos primitivos y no primitivos

3. **Soluciones** – Archivos con código Java que muestran cómo resolver los ejercicios.

---

### Cómo usar este repositorio

1. Clona o descarga el repositorio en tu computadora.
2. Abre un IDE compatible con Java (IntelliJ IDEA, Eclipse, VS Code + Java).
3. Revisa primero teoria.md para comprender los conceptos.
4. Explora los ejemplos en ejemplos/ para ver implementaciones reales y prácticas.
5. Realiza los ejercicios en propuestos.md, escribiendo tu propio código.
6. Verifica tus respuestas comparándolas con soluciones.md.
7. Experimenta cambiando los valores de las variables para reforzar el aprendizaje.
