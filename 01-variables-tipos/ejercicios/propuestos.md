# Ejercicios Propuestos – Variables y tipos de datos en Java

## Instrucciones generales

- Escribe tu código en un solo archivo .java.
- Usa nombres claros para tus variables.
- Imprime los resultados usando System.out.println.
- No uses estructuras de control ni funciones todavía.

### Ejercicio 1 – Declaración y asignación de variables

1. Declara las siguientes variables:

    - Un entero llamado `edad`
    - Un número decimal llamado `estatura`
    - Una letra llamada `inicial`
    - Un valor lógico llamado `esEstudiante`

2. Asigna valores a cada variable.
3. Imprime el valor de cada variable junto con su tipo, por ejemplo:

```java
System.out.println("Edad: " + edad);

```
---

### Ejercicio 2 – Copia de valores de variables primitivos

1. Crea una variable `a` de tipo `int` y asígnale un valor.
2. Crea otra variable `b` y asígnale `a`.
3. Cambia el valor de `b`.
4. Imprime los valores de `a` y `b` y observa qué ocurre.
5. Explica por qué los valores no se afectan mutuamente.

---

### Ejercicio 3 – Valores y tipos

1. Declara variables de diferentes tipos:

    - `int cantidad = 5;`
    - `double precio = 12.50;`
    - `char letra = 'J';`
    - `boolean disponible = true;`
2. Imprime cada variable con un mensaje descriptivo.
3. Cambia los valores y vuelve a imprimirlos para practicar asignaciones.

--- 

### Ejercicio 4 – Operaciones con variables

1. Declara dos enteros `x` y `y` con valores diferentes.
2. Calcula y guarda:
    - Suma (`x + y`)
    - Resta (`x - y`)
    - Producto (`x * y`)
    - División entera (`x / y`)
    - Residuo (`x % y`)
3. Imprime todos los resultados con mensajes descriptivos.
