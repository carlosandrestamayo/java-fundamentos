## 🧩 EJERCICIO 1: GESTIÓN DE VEHÍCULO

### 🎯 Descripción
Desarrollar un programa que permita representar un vehículo y mostrar su información.

### 📋 Requerimientos

1. Crear una clase `Vehiculo` con:
   - marca (String)
   - velocidad (int)

2. Implementar:
   - Un constructor vacío
   - Un constructor con parámetros

3. Crear un método `mostrarDatos()` que imprima la información.

4. Crear un método `acelerar()` que aumente la velocidad en 10.

5. Sobrecargar el método `acelerar(int incremento)`.

---

## 🧩 EJERCICIO 2: CÁLCULO DE ÁREA

### 🎯 Descripción
Crear una clase que permita calcular áreas de diferentes figuras.

### 📋 Requerimientos

1. Crear una clase `Area`.

2. Implementar métodos sobrecargados:
   - `calcularArea(double lado)` → cuadrado  
   - `calcularArea(double base, double altura)` → rectángulo  
   - `calcularArea(double radio, boolean esCirculo)` → círculo  

3. Los métodos deben retornar el área correspondiente.

4. Crear un método estático `mensaje()`.

## 🧩 EJERCICIO 3: GESTIÓN DE EMPLEADO

### 🎯 Descripción
Modelar un empleado y calcular su salario.

### 📋 Requerimientos

1. Crear una clase `Empleado` con:
   - nombre
   - salarioBase

2. Implementar:
   - Constructor con parámetros
   - Método `calcularSalario()`  

3. Sobrecargar el método para incluir bonificación:
   - `calcularSalario(double bono)`

4. Mostrar los datos del empleado.

## 🧩 EJERCICIO 4: SISTEMA DE PRODUCTOS

### 🎯 Descripción
Desarrollar un sistema simple para manejar productos.

### 📋 Requerimientos

1. Crear una clase `Producto` con:
   - nombre
   - precio
   - cantidad

2. Implementar:
   - Constructor vacío
   - Constructor con parámetros

3. Crear un método `calcularTotal()`.

4. Sobrecargar el método para aplicar impuesto.

5. Crear un método estático `mensaje()`.

6. En la clase principal:
   - Crear al menos 2 objetos
   - Mostrar resultados
