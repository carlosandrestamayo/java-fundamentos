# 📘 Programación Orientada a Objetos en Java  
## Abstracción e Interfaces

Este repositorio contiene material educativo sobre conceptos fundamentales de la **Programación Orientada a Objetos (POO)** en Java, enfocados en:

- Abstracción
- Clases abstractas
- Interfaces
- Comparación entre ambos enfoques
- Ejercicios resueltos y propuestos

---

## 🎯 Objetivos de aprendizaje

Al finalizar este material, el estudiante será capaz de:

- Comprender el concepto de abstracción
- Implementar clases abstractas en Java
- Definir e implementar interfaces
- Diferenciar entre clases abstractas e interfaces
- Diseñar soluciones orientadas a objetos
- Tomar decisiones de diseño en problemas reales

---

## 🧠 Contenidos

### 🔹 Abstracción
- Definición
- Ejemplos conceptuales
- Aplicación en Java

### 🔹 Clases abstractas
- Uso de la palabra clave `abstract`
- Métodos abstractos y concretos
- Herencia

### 🔹 Interfaces
- Definición y uso
- Implementación con `implements`
- Interfaces múltiples
- Métodos `default` y `static` (Java 8+)

### 🔹 Comparación
- Diferencias clave entre:
  - Clase abstracta
  - Interfaz

---

## 🧩 Ejercicios incluidos

### ✅ Ejercicios resueltos

1. **Figuras geométricas (Clase abstracta)**
   - Cálculo de áreas
   - Uso de herencia y polimorfismo

2. **Carro eléctrico (Interfaces múltiples)**
   - Implementación de dos interfaces
   - Separación de responsabilidades

---

### 🚀 Ejercicios propuestos

1. **Sistema de empleados**
   - Decidir entre clase abstracta e interfaces
   - Modelado de comportamientos comunes y opcionales

2. **Plataforma de transporte (Caso real)**
   - Diseño de sistema tipo Uber
   - Uso combinado de abstracción e interfaces
   - Enfoque en escalabilidad

## 🛠️ Tecnologías utilizadas

- Java (JDK 8 o superior)
- Programación Orientada a Objetos

---

## ▶️ Cómo ejecutar los ejemplos

1. Asegúrate de tener Java instalado:

```bash
java -version
```

2. Compila el archivo:

```bash
javac Main.java
```

3. Ejecuta el programa:

```bash
java Main
```
