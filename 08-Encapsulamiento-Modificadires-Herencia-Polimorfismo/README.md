# 📘 UNIDAD 07: FUNDAMENTOS DE PROGRAMACIÓN ORIENTADA A OBJETOS EN JAVA

---

## 🎯 Descripción

En esta unidad se introducen los fundamentos de la **Programación Orientada a Objetos (POO)**, abordando los conceptos esenciales para modelar problemas del mundo real mediante clases y objetos en Java.

El enfoque es práctico y progresivo, permitiendo al estudiante construir soluciones completas utilizando principios clave como encapsulamiento, modularidad y reutilización de código.

---

## 🧠 Objetivos de Aprendizaje

Al finalizar esta unidad, el estudiante será capaz de:

- Comprender los conceptos fundamentales de la POO  
- Crear clases y objetos en Java  
- Definir y utilizar atributos y métodos  
- Implementar constructores y sobrecarga  
- Aplicar el uso de la palabra clave `this`  
- Utilizar métodos estáticos (`static`)  
- Comprender la diferencia entre programación estructurada y orientada a objetos  
- Resolver problemas utilizando arreglos dentro de clases
