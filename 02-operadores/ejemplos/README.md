# Ejemplos – Operadores y expresiones en Java


Esta carpeta contiene ejemplos de **código en Java** relacionados con operadores y expresiones.
Cada archivo muestra de forma clara y comentada cómo funcionan los diferentes operadores en Java, incluyendo:

- Operadores aritméticos
- Operadores de asignación
- Operadores relacionales
- Operadores lógicos
- Operadores unarios
- Operador ternario

Estos ejemplos son independientes, compilables y están diseñados con fines educativos.

---

### Contenido:

| Archivo                       | Contenido                                                  |   |                                      |
| ----------------------------- | ---------------------------------------------------------- | - | ------------------------------------ |
| `OperadoresAritmeticos.java`  | Ejemplos de suma, resta, multiplicación, división y módulo |   |                                      |
| `OperadoresAsignacion.java`   | Ejemplos de `=`, `+=`, `-=`, `*=`, `/=`, `%=`              |   |                                      |
| `OperadoresRelacionales.java` | Ejemplos de `==`, `!=`, `>`, `<`, `>=`, `<=`               |   |                                      |
| `OperadoresLogicos.java`      | Ejemplos de `&&`, `\|\| `, `!` y evaluación de cortocircuito |
| `OperadoresUnarios.java`      | Ejemplos de `++`, `--`, `+`, `-` y negación lógica         |   |                                      |
| `OperadorTernario.java`       | Ejemplo del operador ternario `? :`                        |   |                                      |

---

### Instrucciones para usar los ejemplos

1. Clona o descarga el repositorio.
2. Abre un IDE compatible con Java (IntelliJ IDEA, Eclipse, VS Code + Java).
3. Navega a la carpeta 02-operadores-expresiones/ejemplos.
4. Abre cualquiera de los archivos .java.
5. Ejecuta el programa y observa los resultados impresos en consola.
6. Modifica los valores de las variables para experimentar con diferentes expresiones.

---

### Propósito educativo

- Comprender la sintaxis y funcionamiento de cada operador en Java.
- Visualizar cómo cambian los valores de las variables según el operador utilizado.
- Identificar posibles errores comunes y cómo evitarlos.
- Servir como base para ejercicios prácticos y proyectos más complejos.

---

### Recomendaciones

- Compila y ejecuta cada archivo por separado.
- Analiza los comentarios dentro del código.
- Experimenta modificando las variables y operadores.
- Consulta la teoría en teoria.md antes de probar los ejemplos para mejor comprensión.
