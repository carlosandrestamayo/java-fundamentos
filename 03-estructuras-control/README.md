# Unidad 3 – Estructuras de Control en Java

## Descripción general

En esta unidad se estudian las **estructuras de control**, que permiten **dirigir el flujo de ejecución** de un programa en Java.  
Gracias a ellas, un programa puede:

- Tomar decisiones
- Repetir instrucciones
- Controlar cuándo continuar o detener un proceso

Las estructuras de control son fundamentales para construir programas **dinámicos, lógicos y funcionales**.

---

## Contenidos de la unidad

En esta unidad se abordan los siguientes temas:

### 🔹 Estructuras condicionales
- `if`
- `if - else`
- `if - else if - else`
- `switch`
- Uso de `break` y `default`

### 🔹 Estructuras repetitivas
- Ciclo `for`
- Ciclo `while`
- Ciclo `do-while`

### 🔹 Control del flujo
- Uso de `break`
- Uso de `continue`
- Estructuras de control anidadas

---

## 🎯 Objetivos de aprendizaje

Al finalizar esta unidad, el estudiante será capaz de:

- Tomar decisiones usando condiciones lógicas
- Repetir instrucciones de forma controlada
- Comprender cuándo usar cada tipo de estructura
- Analizar y construir algoritmos básicos
- Leer y escribir código con lógica clara y estructurada
