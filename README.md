### 🧠 Descripción del proyecto

Este repositorio contiene material educativo diseñado para **aprender programación en Java desde los fundamentos**, con un enfoque claro en:

- Pensamiento lógico
- Comprensión real del lenguaje
- Buenas prácticas desde el inicio

El contenido está organizado por **unidades temáticas**, cada una con:
- Teoría explicada paso a paso
- Ejemplos en Java
- Ejercicios propuestos
- Soluciones comentadas

Está orientado a **estudiantes universitarios**, autodidactas y docentes que buscan un material **estructurado, progresivo y claro**.

---

### 🎯 Objetivos del repositorio

- Comprender los conceptos fundamentales de la programación
- Aprender Java sin depender solo de la sintaxis
- Desarrollar lógica algorítmica sólida
- Preparar la base para estructuras de datos y programación avanzada

---

### 🧩 Contenidos principales

- Variables y tipos de datos
- Operadores y expresiones
- Estructuras de control
- Arreglos
- Métodos y funciones
- Recursividad
- Ejercicios de razonamiento lógico
